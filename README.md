# Ping Coming Soon

A simple coming soon landing page built with HTML, CSS, and JavaScript.

## Files
- `index.html`: main page markup
- `style.css`: styling
- `script.js`: JavaScript behavior

## Usage
1. Open `index.html` in a browser.
2. Optionally host locally via a static server (e.g., `live-server`).

## Local Server (optional)
- Install `live-server` and run:
  - `npm install -g live-server`
  - `live-server`

## Notes
- This project is a lightweight template for a coming soon page.
- Customize text, style, and script as needed.
